# 🚀 ClipShift

**ClipShift** is a fast video editing & videography marketplace platform that connects clients with professional editors and videographers to deliver high-quality content within hours.

---

## ⚡ Features

### 🎬 Client Features

* Upload videos for editing
* Book videographers for shoots
* Choose editing styles (Basic / Advanced / Premium)
* Real-time chat with professionals
* Fast delivery (6–24 hours)
* Ratings & reviews

### 🎨 Editor Features

* Create profile & portfolio
* Accept/reject jobs
* Upload edited videos
* Earnings dashboard

### 📹 Videographer Features

* Location-based booking
* Schedule shoots
* Upload footage or final videos

---

### Backend

* Node.js
* Express.js
* tRPC

### Database

* MySQL
* Drizzle ORM
